🌐 Sistemas Operacionais

Um sistema operacional (SO) é o software responsável por gerenciar hardware, memória, processos e aplicativos. Ele atua como intermediário entre o usuário e o computador.

📌 Funções principais

  • Gerenciar memória RAM
  • Controlar CPU
  • Executar aplicativos
  • Gerenciar arquivos
  • Garantir segurança

📱 iOS

Sistema da Apple, baseado no Unix (Darwin), usado exclusivamente em iPhones.

🔐 Segurança

  • Sandbox de aplicativos
  • App Store controlada
  • Atualizações rápidas

👍 Vantagens

  • Alta segurança
  • Desempenho fluido
  • Integração Apple

🤖 Android

Sistema do Google baseado no kernel Linux, usado por vários fabricantes.

📱 Características

  • Alta personalização
  • Grande variedade de aparelhos
  • Google Play Store

🐧 Linux

Sistema open-source criado por Linus Torvalds em 1991.

📦 Distribuições

  • Ubuntu
  • Debian
  • Fedora
  • Arch Linux

👍 Vantagens

  • Código aberto
  • Seguro
  • Leve

🪟 Windows

Sistema da Microsoft, o mais usado em computadores pessoais.

📊 Versões

  • Windows 7
  • Windows 10
  • Windows 11

👍 Vantagens

  • Compatibilidade
  • Fácil uso
  • Jogos
